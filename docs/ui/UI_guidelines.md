| 用途 | 值 |
|-----|------|

| 元素间距 | 8px |

## 组件规范

### 按钮

| 类型 | 样式 | 用途 |
|-----|------|------|
| Primary | 主色背景+白字 | 主要操作 |
| Secondary | 边框+主色字 | 次要操作 |
| Text | 无背景+主色字 | 轻量操作 |
| Danger | 红色 | 危险操作 |

### 表单

- 标签：在输入框上方
- 必填：标签后加红色*
- 错误：红色边框+下方提示

### 卡片

- 圆角：8px
- 阴影：0 2px 8px rgba(0,0,0,0.1)
- 内边距：16px

## 页面布局

```
┌─────────────────────────────┐
│         Header (60px)        │
├─────────────────────────────┤
│ Sidebar │     Content       │
│ (200px) │                   │
└─────────────────────────────┘
```

## 响应式断点

| 断点 | 宽度 | 布局调整 |
|-----|------|---------|
| Mobile | <768px | 单列，隐藏侧栏 |
| Tablet | 768-1024px | 侧栏折叠 |
| Desktop | >1024px | 完整布局 |
